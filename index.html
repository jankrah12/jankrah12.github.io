DaMixhub is a music platform where users stream and download mixtapes. I created it in 2012 after I could not find a suitable option to stream mixtapes I loved. I set out to learn java and put together a simple streaming app. It slowly grew in popularity and lots of DJs and artists started to contact me. It was time to go big time a couple years later.

I span up a few more servers and hired some freelancers to help with things like day-to-day administration, social media - oh and I also needed a system admin. That was the fun stuff. The not so fun stuff was all the legal and business stuff like setting up an LLC, taxes and accounting. In 2016 I had to make the decision to quit my job as a systems engineer. I developed a website to go along with the mobile apps. The site was developed from the ground up using HTML and javascript for frontend, php and mysql for backend. I also developed admin portals for managing things like approving user submissions and various automation tasks.

In 2018, I fell in love with React Native and decided to re-engineer my app using the the new technology. This was a huge undertaking but it provided me the benefit of developing a cross-platform app amongst other benefits. The biggest challenge was migrating data from native to React Native and making that transition seem-less for the users. In the end, the feedback I received was overwhelming positive and the apps are easier to monitor, debug and add features

Invites is a dating app where users setup dates and connect on mutual interest. To be honest, Invites is more of a pet project than anything. I have always wanted to create a dating app - why not? Every one is doing it! When learning React Native, I wanted to push myself and figured that creating an actual product will be the best way to learn.

So I designed a quick logo and UI and set out to to conquer the world. The backend was solely developed on Google Firebase because it quick and easy to get going. The app is loaded with features such as profile setup, dates and event setup and of course messaging. I got a simple website from Wix for this one and I was ready for launch. A big part of the project for me was learning how to launch and market an app platform.

So I sent out a press release, contacted a few sites and blogs. For marketing, I setup advertising campaigns on Facebook, Snapchat and Google. It was a fun experience and I learned lots of lessons. The biggest being that it takes a lot of money and more than a one-man-band to get traction in todays app market.
